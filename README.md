# CampusFoods Pro

An AI-powered multi-instance food ordering and queue optimization ecosystem for campuses.

## Features
- RFID + QR Login
- AI Queue Balancer
- IoT Food Sensors
- Smart Waste Management
- Dynamic Theming & Animations
- Group Ordering System

## Tech Stack
- Frontend: Next.js + Framer Motion + Lottie
- Backend: Node.js + WebSockets + Express
- AI: Python (ML Models)
- Database: MongoDB
- IoT: MQTT Integration

cd client
npm install
npm run dev
